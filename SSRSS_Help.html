What is SSRSS

Overview
SSRSS - The Super Simple RSS syndication software.  Ok, so you've run the installation and loaded up SSRSS, what the heck does this whacked out interface mean?  Well it's really simple, trust me.  SSRSS allows you to create RSS files, and RSS files are organized into entries.  So First of All, you're going to want to create a new entry.  Once you have entered the information you want in your entry, add it to the channel.  Repeat these previous two steps as many times as you like.  Now your channel has content!  Let's set some global channel options now by adjusting rss options.  Ready to save your .rss file?  Congratulations, you now have a RSS feed you can upload to your website.  See, I told you it was simple!

Load In an RSS File
To load in a file, simply click the "load RSS file" button.  Select the file you want to load, and click open.  Voila!  You have loaded a file.  *Note - You can only load RSS files that have been created by SSRSS.

Save Out an RSS File
To save a file, simple click the "save" button.  Select a file you want to save over, or enter a filename into the filename box.  Then click save.  Voila! You have saved a file.

Create a New Entry
To create a new entry, simply click on the "new entry" button.  If you have any unsaved data in the entry fields you will be prompted whether to add it or not. 

Add Entry to Channel
To add the current entry information to the channel, simply click the "add entry" button.

Delete an Entry
To delete an entry, simply highlight the entry to be deleted in the list of items, and click "delete selected entry".

Adjust RSS Options
To adjust RSS options (set channel title, etc...) simple click on the "Rss Options" button, fill in whichever fields you like, and click ok.

Create HTML Link
To create an HTML link in your entry text, simply highlight the word you wish to become an HTML link, then click "create hyperlink".  A dialog will pop up, enter the website address you wish to link to here. i.e. (http://www.hunterdavis.com)

Clear Entry Boxes
To clear the entry boxes, simple click the "clear" button.  If you have any unsaved data in the entry fields you will be prompted whether to add it or not.
